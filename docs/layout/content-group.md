> [Content grid][content grid] is for displaying grids of items in a responsive format.

[Content grid][content grid] uses flexbox for layout meaning the each element will be aligned to the top of its row.  

# Two up
```html
responsive: ['Phone', 'Tablet', 'Desktop']
---
<div class="content-group content-group--grid-2up">
  <div class="content-group__item"><img alt="Image description" src="/exfoliation-design-system/assets/core/image-300x300.gif" /></div>
  <div class="content-group__item"><img alt="Image description" src="/exfoliation-design-system/assets/core/image-300x300.gif" /></div>
  <div class="content-group__item"><img alt="Image description" src="/exfoliation-design-system/assets/core/image-300x300.gif" /></div>
  <div class="content-group__item"><img alt="Image description" src="/exfoliation-design-system/assets/core/image-300x300.gif" /></div>
</div>
```
